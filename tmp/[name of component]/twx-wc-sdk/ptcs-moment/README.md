# ptcs-moment

## Overview

- Reimports `moment.js` module in SDK  
- Reimports global `window.moment` as `moment` in Mashup Builder

## Usage Examples

```js
import moment from 'ptcs-moment/moment-import.js'
```

```json
{

    "dependencies": {
        "ptcs-moment": "file:../ptcs-moment"
    }

}
```
